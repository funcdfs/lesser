## 测试命令

### 1. 用户注册

```bash
grpcurl -plaintext -d '{"username": "testuser999", "email": "test999@example.com", "password": "Test123456"}' localhost:50053 gateway.GatewayService/Register
```

### 2. 用户登录

```bash
grpcurl -plaintext -d '{"username": "testuser999", "password": "Test123456"}' localhost:50053 gateway.GatewayService/Login
```

### 3. 创建会话

```bash
grpcurl -plaintext -d '{"type": 0, "name": "test chat", "member_ids": ["<user1_id>", "<user2_id>"], "creator_id": "<user1_id>"}' localhost:50052 chat.ChatService/CreateConversation
```

### 4. 发送消息

```bash
grpcurl -plaintext -d '{"conversation_id": "<conv_id>", "sender_id": "<user_id>", "content": "Hello!", "message_type": "text"}' localhost:50052 chat.ChatService/SendMessage
```

### 5. 标记单条消息已读

```bash
grpcurl -plaintext -d '{"message_id": "<msg_id>", "user_id": "<user_id>"}' localhost:50052 chat.ChatService/MarkAsRead
```

### 6. 标记会话已读

```bash
grpcurl -plaintext -d '{"conversation_id": "<conv_id>", "user_id": "<user_id>"}' localhost:50052 chat.ChatService/MarkConversationAsRead
```

### 7. 获取未读数

```bash
grpcurl -plaintext -d '{"user_id": "<user_id>", "conversation_ids": ["<conv_id>"]}' localhost:50052 chat.ChatService/GetUnreadCounts
```
