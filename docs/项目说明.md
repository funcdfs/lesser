# Lesser 项目

全栈社交应用：Flutter + Django + APISIX

## 快速开始

```bash
./dev.sh          # 启动
./dev.sh logs     # 日志
./dev.sh stop     # 停止
```

## 服务地址

- Django Admin: http://localhost:8001/admin/
- APISIX Dashboard: http://localhost:9000/
- API Gateway: http://localhost:9080/

管理员: `funcdfs` / `fw142857`

├── backend/django_code/    # Django API
