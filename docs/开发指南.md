## 新增 API 流程

### 1. Django 后端

```bash
# 位置: backend/django_code/app/<模块>/
```

**添加模型** (`models.py`):
```python
class Article(models.Model):
    title = models.CharField(max_length=200)
    content = models.TextField()
    created_at = models.DateTimeField(auto_now_add=True)
```

**添加序列化器** (`serializers.py`):
```python
class ArticleSerializer(serializers.ModelSerializer):
    class Meta:
        model = Article
        fields = '__all__'
```

**添加视图** (`views.py`):
```python
class ArticleViewSet(viewsets.ModelViewSet):
    queryset = Article.objects.all()
    serializer_class = ArticleSerializer
```

**注册路由** (`urls.py`):
```python
router = DefaultRouter()
router.register('articles', ArticleViewSet)
urlpatterns = [path('', include(router.urls))]
```

**迁移数据库**:
```bash
docker exec django python manage.py makemigrations
docker exec django python manage.py migrate
```

### 2. Flutter 前端

```bash
# 位置: frontend/lib/
```

**添加模型** (`data/models/article.dart`):
```dart
class Article {
  final int id;
  final String title;
  final String content;
  
  Article.fromJson(Map<String, dynamic> json)
      : id = json['id'],
        title = json['title'],
        content = json['content'];
}
```

**添加 API 调用** (`data/repositories/article_repository.dart`):
```dart
class ArticleRepository {
  Future<List<Article>> getArticles() async {
    final response = await dio.get('/api/content/articles/');
    return (response.data as List)
        .map((e) => Article.fromJson(e))
        .toList();
  }
}
```

## 网关路由

通过 APISIX Dashboard (http://localhost:9000/) 配置路由：

1. 创建上游 (Upstream): 指向 `django:8000`
2. 创建路由 (Route): 配置 URI 和上游

或使用 API:
```bash
curl -X POST http://localhost:9180/apisix/admin/routes \
  -H "X-API-Key: fw142857" \
  -H "Content-Type: application/json" \
  -d '{
    "uri": "/api/*",
    "upstream": {"nodes": {"django:8000": 1}}
  }'
```
